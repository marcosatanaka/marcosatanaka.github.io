What’s New in Play for iOS 18, iPadOS 18, and tvOS 18

Play’s upcoming update will support the latest features introduced in iOS 18, iPadOS 18, and tvOS 18. It deeply integrates with Spotlight to get ready for the upcoming Apple Intelligence semantic index, offers controls for the redesigned Control Center, and includes new app icons that match your Home Screen style.

It also brings new forms of interaction via App Shortcuts that allows you to add and retrieve data using Siri, Spotlight, the Shortcuts app, and more. The new Translate APIs are being used to translate video descriptions to the user’s preferred language. And, the Apple TV app now features the new floating sidebar with a gorgeous translucent background, matching other system apps like TV.

Spotlight

Play’s content is now searchable in Spotlight. This will be especially useful for the upcoming semantic index and Apple Intelligence features.

Translate

The video description can now be translated using Apple’s built-in Translate framework.

Apple TV tvOS 18

Play’s sidebar is now built using the new floating sidebar API, allowing users to take advantage of the same appearance and functionality already available in other system apps, such as TV.
