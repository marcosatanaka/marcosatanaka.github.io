What’s New in MusicBox

Version 1.6.0

Released September 14, 2025


MusicBox now features a modernized UI for Liquid Glass and iOS 26, updated app icons, and intelligent features that help you throughout the app:

  • Fresh design: Deeper separation between content and navigation controls, buttons that animate and shift shape as you move between different views, and playful navigation transitions for a polished user experience. Music lists now include a subtitle with relevant metadata, and the search bar has been relocated to the bottom for easier access. Modal sheets throughout the app have been redesigned to only take up the space they actually need, making them less likely to get in the way of other content you’re viewing.
  • Updated app icons: MusicBox now supports every icon appearance for customizing your Home Screen.
  • Tag and icon suggestions: On supported devices, you can now get AI-generated tag and icon ideas for your content.

  • Redesigned start screen and sidebar on iPhone and iPad.

Version 1.4.3

Released October 23, 2024


This update includes several improvements and features based on user feedback from the past weeks:

  • Shortcuts
    • Add Music now supports the same set of services supported by the main app.
    • Add Music Manually now includes track count, duration, editorial notes, and rating as fields.
    • Find Music now supports finding items by their ID.
    • Show Music can be customizable to either open the item in its source app or view it in MusicBox.
    • Add or Edit Notes action was introduced to manipulate notes for an item.
    • Mark as watched now properly updates the Date Played attribute.
  • Services
    • Better support for albums, tracks, and artists from Songwhip and AllMusic.
    • Podcasts from Apple Podcasts will now get the “Podcast” type.
  • User Interface
    • New option to organize new music by genre on the start screen. To configure, go to Settings → Start Screen.
    • Separate sections for albums and singles when adding via search.
    • The full album title will now be shown in the details view.
    • Temporarily remove support for widget accented mode due to a system issue that stops links from opening their associated items.
  • Auto-Tagging
    • Editing an Auto-Tagging condition and changing the 'When' attribute no longer unnecessarily clears the 'Contains' field.

Version 1.2.4

Released Sep 25, 2023

  • Widgets have been reorganized to offer unified options for the Lock Screen and Home Screen. There are now three types of widgets: List, Single Item, and Add Music. I also added more sizes, specially the smaller ones, allowing for more flexible widget setups. Large and extra-large interactive widgets now have two new background colors: Black and System, which follows the system’s defaults for light and dark mode. They also offer a Minimal UI option that hides every UI element.
  • Customizable Start Screen and Sidebar: You can now customize which sections are visible in the start screen/sidebar. Show or hide tags, smart search, a new list of actions for quickly adding music, and even new sections that automatically groups music by artist and record label. Go to Settings → Start Screen to configure.
  • MusicBox can now optionally group items by artist and label, allowing you to easily catch up on new music from a particular creator. Artists and labels with new music will be shown in the start screen/sidebar and are sorted by the count of new music and then by name. Go to Settings → Sidebar to activate.
  • New Super Dense Grid layout option that shows more music per page. If you’re on the iPad, try pairing it with a two-column layout for a more focused music list. Go to Settings → Music List to configure.
  • Smart Search now supports filtering by multiples artists, genres, and labels when using a comma to separate the name (e.g., boygenius,oasis).
  • The Add to MusicBox share sheet action now supports adding music manually. When adding music from an app that MusicBox can’t get the metadata automatically, the window will allow you to manually type the missing metadata.
  • The music’s star rating is now present in the response of the Find Music shortcut action, and also available as an attribute for filtering.
  • The Record Label can now be shown as a metadata option for the list and grid layout.

Version 1.2.3

Released Sep 13, 2023


MusicBox 1.2.3 brings new features for iOS and iPadOS 17 including interactive widgets, support for StandBy mode on the iPhone, support for Lock Screen widgets on the iPad, and more:

  • Interactive Widgets: Edit a large or extra-large widget to customize and configure interactive Tags and Smart Searches.
  • Support for the iPhone’s new StandBy mode.
  • Support for Lock Screen widgets on the iPad.
  • MusicBox now shows tips about features you may not know about.
  • Improvements and bug fixes:
    • New Add Music Manually shortcut action allows you to add music manually by filling in all the required metadata.
    • Added the ability to open MusicBox without viewing a random item when tapping the Random Item Lock Screen widget.
    • The Played section now always sorts items by the date played attribute.
    • Improved dark mode support and legibility for tag colors.
    • MusicBox now remembers the state of the Tags list (shown/collapsed).
    • Fixed a bug that could cause the creation of Apple Music playlists to fail.
