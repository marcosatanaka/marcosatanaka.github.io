Version 1.4.3

Released October 22, 2024


This update includes several improvements and features based on user feedback from the past weeks:

  • Shortcuts
    • Add Music now supports the same set of services supported by the main app.
    • Add Music Manually now includes track count, duration, editorial notes, and rating as fields.
    • Find Music now supports finding items by their ID.
    • Show Music can be customizable to either open the item in its source app or view it in MusicBox.
    • Add or Edit Notes action was introduced to manipulate notes for an item.
    • Mark as watched now properly updates the Date Played attribute.
  • Services
    • Better support for albums, tracks, and artists from Songwhip and AllMusic.
    • Podcasts from Apple Podcasts will now get the “Podcast” type.
  • User Interface
    • New option to organize new music by genre on the sidebar. To configure, go to Settings → Sidebar.
    • Ability to adjust the sorting order directly from the toolbar.
    • Separate sections for albums and singles when adding via search.
    • Notes can now be added or edited inline in the details pane.
  • Auto-Tagging
    • Editing an Auto-Tagging condition and changing the 'When' attribute no longer unnecessarily clears the 'Contains' field.

Version 1.3.5

Released June 5, 2024

  • Pinned Tags and Smart Searches now sync across platforms, and there’s no limit to the number of items you can pin.
  • The notes text editor now features live markdown syntax highlighting and improved controls.
  • Search now offers title, artist, notes, and tag as categories for more granular control.
  • New button in the item’s context menu to quickly filter by that artist.
  • New button in the File menu to quickly save music from the clipboard.
  • Fixed an issue that could cause state restoration to not load the previously selected Smart Searches on launch.

Version 1.3.4

Released May 10, 2024

  • Included the option to edit the artist name.
  • Fixed an issue where the URL sent through the musicbox://add?url= URL Scheme was replaced by the URL in the clipboard.

Version 1.3.3

Released Apr 22, 2024

  • Fixed an issue where the app could crash when launched by users with numerous tags and tag folders.
  • Fixed an issue where double-clicking an item would unexpectedly open a window displaying metadata.

Version 1.3.2

Released Apr 20, 2024

  • The music list now indicates which items have a star rating or notes.
  • Included buttons to help with Markdown formatting while editing a note.
  • Support undo (⌘ Z) and redo (⇧ ⌘ Z) functions for operations with music items.
  • Tag and Smart Search folders can now be edited and deleted from the sidebar.
  • Tag folders now display the item count.
  • Improved support for music from Mixcloud.
  • Added an option to remove the Add to MusicBox button from the menu bar.
  • The Find Music shortcut action now supports filtering by URL.
  • New Music Details section in Settings that allows customizing information displayed within the music details panel.

Version 1.3.1

Released Apr 1, 2024

  • The list of supported streaming services has been significantly expanded. If MusicBox couldn't fetch metadata for a music you tried to add before, there's a good chance it can now handle it.
  • MusicBox now also supports a wide range of content types, including articles, posts, and so on. This opens up various possibilities, such as saving links to podcast episodes, album reviews, articles listing top albums, and so on.
  • New menu bar button for quickly adding music when the app is not active.
  • Editorial and notes are now embedded in the music details panel for a more streamlined experience.
  • Settings now embraces a UI design more aligned with the platform.

Version 1.3.0

Released Mar 20, 2024

  • Create folders for Tags and Smart Searches to better organize your music collection.
  • Fixed an issue that could cause the Played list to not be sorted in reverse chronological order.

Version 1.2.9

Released Mar 14, 2024

  • Added support for Tidal albums, artists, and singles.
  • From this update forward, the Duration attribute will be included for albums, singles, and playlists on compatible platforms. It is also available as a Smart Search and Auto-Tagging condition.
  • You can now create rules for Auto-Tagging based on the year of release.
  • Fixed an issue that prevented Shortcuts who receive parameters from the Share Sheet from executing after sharing a music item.

Version 1.2.8

Released Mar 6, 2024

  • You can now view the number of items on each Smart Search
  • Play a random new item when pressing and holding a Tag or Smart Search

Version 1.2.7

Released Feb 17, 2024

  • Pin up to two Tags or Smart Searches at the sidebar for quick access.
  • Smart Search now supports filtering by URL.
  • Smart Search now supports filtering items that do not include a specific tag.
  • Smart Search now supports customizing the criteria between Contains and Doesn’t Contains when filtering by title, artist, genre, label, notes, or URL.
  • Drag & Drop improvements: Drag a tag into an item to assign.
  • New Organize menu now aggregates access to Tags, Auto-Tagging and Smart Search.
  • Fixed an issue that could cause some specific albums to not be added.

Version 1.2.6

Released Dec 12, 2023

  • It is now possible to show the Date Played as Metadata #1 or Metadata #2 in the List and Grid layout.
