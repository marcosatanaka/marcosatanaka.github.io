What’s New in MusicBox

Version 1.3.1

Released Mar X, 2024


• New menu bar buttom for quickly adding music when the app is not active.

• Editorial and notes are now embedded in the music details panel for a more streamlined experience.

• Settings now embraces a UI design more aligned with the platform.

Version 1.3.0

Released Mar 20, 2024


• Create folders for Tags and Smart Searches to better organize your music collection.

• Fixed an issue that could cause the Played list to not be sorted in reverse chronological order.

Version 1.2.9

Released Mar 14, 2024


• Added support for Tidal albums, artists, and singles.

• From this update forward, the Duration attribute will be included for albums, singles, and playlists on compatible platforms. It is also available as a Smart Search and Auto-Tagging condition.

• You can now create rules for Auto-Tagging based on the year of release.

• Fixed an issue that prevented Shortcuts who receive parameters from the Share Sheet from executing after sharing a music item.

Version 1.2.8

Released Mar 6, 2024


• You can now view the number of items on each Smart Search

• Play a random new item when pressing and holding a Tag or Smart Search

Version 1.2.7

Released Feb 17, 2024


• Pin up to two Tags or Smart Searches at the sidebar for quick access.

• Smart Search now supports filtering by URL.

• Smart Search now supports filtering items that do not include a specific tag.

• Smart Search now supports customizing the criteria between Contains and Doesn’t Contains when filtering by title, artist, genre, label, notes, or URL.

• Drag & Drop improvements: Drag a tag into an item to assign.

• New Organize menu now aggregates access to Tags, Auto-Tagging and Smart Search.

• Fixed an issue that could cause some specific albums to not be added.

Version 1.2.6

Released Dec 12, 2023


• Streamlined navigation by opening associated elements in sheets within the same window, rather than launching multiple new windows.

• Added Set and Mix to the list of possible item types.

• Added Date Played as a possible attribute for filtering and sorting music in the Find Music shortcut action.

• It is now possible to show the Date Played as Metadata #1 or Metadata #2 in the List and Grid layout.

• You can now select and copy the album title and artist name in the album details view.

• Fix: Importing music from a playlist no longer duplicates existing items.

• Fix: Marking an item as played from the Batch Edit view now sets the Date Played attribute.

Version 1.2.5

Released Nov 13, 2023


• The large widget now shows 16 albums when configured with the Minimal UI option.

Version 1.2.3

Released Sep 26, 2023


• Interactive Widgets: Edit a large or extra-large widget to customize and configure interactive Tags and Smart Searches.

• New Add Music Manually shortcut action allows you to add music manually by filling in all the required metadata.

• Command-F now activates the search field.

• The Played section now always sorts items by the date played attribute.
